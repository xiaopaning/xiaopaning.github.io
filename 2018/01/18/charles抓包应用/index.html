charles抓包应用

Veröffentlicht am 2018-01-18

本篇是关于 charles 在 mac 下调试 iphone H5页面。

  • 第一：先了解下 charles 是干啥的
    抓包抓包抓包！！！！这锅我不背！

  • 第二： 首先下载 charles 。有破解版的更好，下载之后打开。

  • 第三：charles 设置步骤：

  1. 菜单栏 —-> proxy —-> proxy settings

  2. 菜单栏 —-> proxy —-> SSL proxying settings

这一步是为了能抓到https请求。

  1. 菜单栏 —-> help —-> install charles Root certificate on a Mobile Device Or Romate Browers

    然后会出现一个弹框，提示说是在手机上安装 certificate

  • 第四：ios 设置：

    1. 将手机网络跟电脑链接同一个网络，在网络中设置代理

    在红框中填写对应信息。其中代理服务器，可以通过电脑charlen，菜单栏 —-> help —-> Local IP Address 查看。
    端口后，与第三步（1）图中保持一致
    ⚠️ 如果 charles 出现如下弹框

    ![image](https://wx4.sinaimg.cn/mw690/006qGo4Oly1fnl0wpwof5j317a08yaby.jpg)

点击 allow

    1. 打开手机浏览器，输入下列地址：
      http://www.charlesproxy.com/getssl
      下载下来，直接安装。如下图：

      ![image](https://wx4.sinaimg.cn/mw690/006qGo4Oly1fnl0wr9xllj30k20yigrd.jpg)

      此图片中，红框应该为 <安装>，点击安装。成功之后，就会跟上边图片显示一致。

    2. 像iphone6以上或许会出现，手机连不上网了，原因不知道为啥，不过可以通过下列方法解决。

    手机 设置 —-> 通用 —-> 关于本机 —-> 证书信任设置（在最底部） ，进入之后，标绿色，ok了。

    现在就可以成功链接了。很完美。

    多嘴一句，我们安装的证书，位置在：

    设置 ----> 通用 ----> 关于本机 ----> 证书信任设置
