hexo+github

hexo 搭建个人博客

hexo

快速、简洁且高效的博客框架

在浏览器中打开 http://localhost:4000/ 链接
若页面显示了hexo，则表示成功，可以进行下来的步骤

github page

接下来在GitHub中创建一个仓库，复制粘贴ssh，格式如下

git@github.com:github_name/repository_name.github.io.git

打开本地blog下的_config.yml，将deploy补充完整

注意此处分号前边无空格，后边有空格，否则会报错

好了，大功告成。下来就开始玩自己的博客了

创建好之后，进入

source --> _posts --> fisrt_blog.md

使用makedown 语句，写入自己的博客

hexo g 
hexo d

完成！！
打开 https://repository_name.github.io 就可以之间观看了

改变hexo的主题

themes --> landscape 是默认主题

更换主题需要做两步
第一步 安装主题

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

第二部 配置
打开_config.yml，作如下修改

theme: yilia

本地起一个服务
hexo s
主题更新了，那就搞定了，接着：

hexo g
hexo d

完成GitHub page 上的更新
