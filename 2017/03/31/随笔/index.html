随笔

Veröffentlicht am 2017-03-31

outline 移除当选中input元素的时候会出现的状态线

input{outline:none}

contenteditable p标签可编辑

<p contenteditable="true">可编辑</p>

webkit-playinline 视频在页面中播放，而不是全屏

<video webkit-playinline></video>

clearfix 清除浮动

.clearfix:after{
    content:'';
    display: block;
    clear: both;
    visibility: hidden;
}

.clearfix{
    zoom: 1
}

user-select禁止用户选中文本

div{
    user-select: none
}

清除手机tap事件后element时候出现的一个高亮

{
    -webkit-tap-highlight-color:rgba(0, 0, 0, 0)
}

:target选择器

URL 带有后面跟有锚名称 #，指向文档内某个具体的元素。这个被链接的元素就是目标元素，如：

<a href="#content1">a标签</a>

这个选择器，完全可以实现tab切换了

linear-gradient可实现背景渐变
观察鼠标移入效果

css 
    .btn {
        margin: 50px auto;
        width: 200px;
        line-height: 64px;
        text-align: center;
        color: #fff;
        font-size: 24px;
        cursor: pointer;
        background: linear-gradient(#024e8a 0%, #2d80c1 50%, #83c7fd 100%);
        background-size: 100% 150%;
        background-position: 0 0;
        transition: .3s;
    }
    .btn:hover {
        background-position: 0 75%;
    }
 html
      <div class="btn">Confirm</div>

linear-gradient 也可以做背景渐变的动画效果，可以发挥自己的想象力噢，作出酷炫拽的效果。

在这里给大家推荐一个网站，可以找到自己想要的效果，直接copy css: https://webgradients.com
