TypeScript

Veröffentlicht am 2017-05-25

最近在用webpack以及react，经常会出现一些类似.ts的文件，不知道是什么鬼，也没有研究过，虽然不知道是干啥的，但是webpack以及react还是可以顺利写下去的，不影响大局。纠结一番之后，还是保持我这颗严谨的心，拿下它（.ts）。

TypeScript 是一种由微软开发的自由和开源的强类型语言，强类型语言的优势在于静态类型检查，是JavaScript的一个超集，扩展了JavaScript的语法。

TypeScript 安装

首先安装npm
其次安装typeScript

npm install -g typescript

安装完成之后，就可以使用 TypeScript 编辑器了。
首先创建一个文件，比如index.ts, 运行下列代码

tsc index.ts

如若成功，同级目录下会生成一个index.js文件，若没有成功，卸载重新安装，在进行上一步骤。

TypeScript 开发

  1. 首先来按照自然规律，跟世界打个招呼
    创建index.html文件，写入：

    <!DOCTYPE html> 
<html> 
<head> 
<meta charset="utf-8"> 
<title>Learning TypeScript</title>
</head> 
<body> 
    <script src="hello.js"></script>
</body> 
</html>

    创建hello.ts文件，写入：

    console.log('Hello, TypeScript')

    运行：

    tsc hello.ts

    观察是否有生成 .js 文件，打开浏览器控制台是否有打印。
    2.尝一尝它的类型批注

TypeScript 通过类型批注提供静态类型以在编译时启动类型检查。这是可选的，而且可以被忽略而使用 JavaScript 常规的动态类型。

创建一个test.ts文件，写入：

function Add(x: number, y: number): number {
    return left + right;
}
Add(1, 1);
Add(1, 'abc');

观察一下编辑器控制台，以及浏览器控制台有何区别。浏览器控制台会正常打印出结果，而编辑器会提示报错。
3.品味一样的实例

创建interface.ts文件，并引入到index.html文件中，写入：

interface Shape {
    name: string;
    width: number;
    height: number;
    color: string;
}

function area(shape : Shape) {
    var area = shape.width * shape.height;
    return "I'm " + shape.name + " with area " + area + " cm squared";
}

console.log( area( {name: "rectangle", width: 30, height: 15} ) );
console.log( area( {name: "square", width: 30, height: 30, color: "blue"} ) );

试着用专业知识搞定它……

I'm rectangle with area 450 cm squared
I'm square with area 900 cm squared

若上述例子中，增加下面console，结果如何呢？

console.log( area( {width: 30, height: 15} ) );

报错！！！

➜  tyscript tsc interface.ts
interface.ts(15,20): error TS2345: Argument of type '{ width: number; height: number; }' is not assignable to parameter of type 'Shape'.
Property 'name' is missing in type '{ width: number; height: number; }'.

然而浏览器控制台照样会打印出结果：

I'm rectangle with area 450 cm squared
interface.js:6 I'm square with area 900 cm squared
interface.js:7 I'm undefined with area 450 cm squared

5.高深莫测的类

为了体现我努力想学会使用这个类，特地新创建了一个class.ts，来表示在我心中的重要位置。并写入：

class Shape {
    area: number;
    color: string;
    constructor ( name: string, width: number, height: number ) {
        this.area = width * height;
        this.color = "pink";
    };
    shoutout() {
        return "I'm " + this.color + " " + this.name +  " with an area of " + this.area + " cm squared.";
    }
}
var square = new Shape("square", 30, 30);
console.log( square.shoutout() );
console.log( 'Area of Shape: ' + square.area );
console.log( 'Name of Shape: ' + square.name );
console.log( 'Color of Shape: ' + square.color );
console.log( 'Width of Shape: ' + square.width );
console.log( 'Height of Shape: ' + square.height );

运行之后，注意一下编辑器控制台，以及浏览器控制台。好好🔥🔥脑吧。哇哈哈！！
明白了之后，将以下代码添加进去：

class Shape3D extends Shape {
    volume: number;
    constructor ( public name: string, width: number, height: number, length: number ) {
        super( name, width, height );
        this.volume = length * this.area;
    };
    shoutout() {
        return "I'm " + this.name +  " with a volume of " + this.volume + " cm cube.";
    }
    superShout() {
        return super.shoutout();
    }
}
var cube = new Shape3D("cube", 30, 30, 30);
console.log( cube.shoutout() );
console.log( cube.superShout() );
