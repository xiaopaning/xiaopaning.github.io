Express

Veröffentlicht am 2017-06-09

一个不会写路由的前端攻城师不是一个好的女程序媛！！！犹豫了一大早，不知道该如何写起，思来想去，决定就从Express说起，从它结束。

Express是一个高度包容、快速而极简单的node框架。

安装Express

⚠️安装并使用Express之前，默认是已经安装了node。

首先创建一个空文件，安装express框架

$ mkdir myapp
$ cd myapp
$ npm init
$ npm install
$ npm install express --save-dev

Hello world

在 myapp 文件下，创建一个app.js文件， 并写入：

var express = require('express');
var app = express();

app.get('/', function (req, res) {
  res.send('Hello World!');
});
// 监听3000端口
app.listen(3000, function () {
  console.log('Example app listening on port 3000!');
});

命令行中运行

$ node app.js

打开浏览器，输入 http://localhost:3000/ 查看结果。好了，成功入门Express。

Express 目录架构生成器

首先 express-generator

$ npm install express-generator g  // 全局安装
$ express --view=pug myapp    // 创建一个命名为myapp的文件夹 
$ cs myapp
$ npm install
$ DEBUG=myapp:* npm start //mac 或者 Linux 下运行
> set DEBUG=myapp:* & npm start // win下运行

基本路由

路由采用以下结构：

app.METHOD(PATH, HANDLER)

app是express的实例，如上例中的var app = express(),

METHOD是请求的方法，

PATH是服务器上的路径，

HANDLER是路由匹配正确之后的执行函数.

1. method

在看上面的例子，

app.get('/', function (req, res) {
  res.send('Hello World!');
});

get就是请求的方法，function就是要处理的函数。
我们知道的，http的请求方式很多的，如get/post/head/delete/options/patch/put等。

上述 🌰 是一个 get 请求。post 呢？

app.post('/', function (req, res) {
  res.send('POST request to the homepage');
});

其实写法都是一样一样的。

看一个特殊的：

app.all('/', function (req, res) {
  res.send('POST request to the homepage');
});

用法如其名，就是不管什么请求方式，他都支持。

2. PATH

path, 定义可以在其中提出请求的端点，路由路径可以是字符串、字符串模式或正则表达式。

🌰 1 ：

app.get('/about', function (req, res) {
  res.send('about');
});

🌰 2 ：

app.get('/random.text', function (req, res) {
  res.send('random.text');
});

🌰 3 ：

app.get('/ab?cd', function(req, res) {
  res.send('ab?cd');
});

path 是 abcd 或者 acd

🌰 4 ：

app.get('/ab+cd', function(req, res) {
  res.send('ab+cd');
});

path 是 abcd、abbcd 或者 abbbcd

🌰 5 :

app.get('/ab*cd', function(req, res) {
  res.send('ab*cd');
});

path 是 abcd、 abxcd、 abjgejkcd 或者 ab12cd

🌰 6 :

app.get('/ab(cd)?e', function(req, res) {
  res.send('/ab(cd)?e');
});

path 是 abe 或者 abcde

🌰 7 :

app.get(/a/, function(req, res) {
  res.send('/a/');
});

path 包含a的

3. headle

🌰 1: 处理一个函数
app.get(‘/example/a’, function (req, res) {
res.send(‘Hello from A!’);
});

🌰 2:处理多个函数，确保指定了 next。如：

 app.get('/example/b', function (req, res, next) {
  console.log('the response will be sent by the next function ...');
  next();
}, function (req, res) {
  res.send('Hello from B!');
});

🌰 3：处理一组函数：

 var cb0 = function (req, res, next) {
  console.log('CB0');
  next();
}
var cb1 = function (req, res, next) {
  console.log('CB1');
  next();
}
var cb2 = function (req, res) {
  res.send('Hello from C!');
}
app.get('/example/c', [cb0, cb1, cb2]);

🌰 4: 处理一组和独立的函数

 var cb0 = function (req, res, next) {
  console.log('CB0');
  next();
}
var cb1 = function (req, res, next) {
  console.log('CB1');
  next();
}
app.get('/example/d', [cb0, cb1], function (req, res, next) {
  console.log('the response will be sent by the next function ...');
  next();
}, function (req, res) {
  res.send('Hello from D!');
});

响应的方法

响应的方法可以参考：
http://expressjs.com/zh-cn/4x/api.html

这里列举一些常用的api

res.end() 结束响应
res.redirect() 重定向请求
res.render() 渲染视图
res.send() 发送各种类型的响应
res.json() 发送json响应
res.jsonp() 在jsonp的支持下发送json响应

在 Express 中提供静态文件

使用中间件函数 express.static()，例如： 

app.use(express.static('public'));
app.use(express.static('files'));

要为 express.static() 函数提供的文件创建虚拟路径前缀（路径并不实际存在于文件系统中），请为静态目录指定安装路径，如下所示：

app.use('/static', express.static('public'))

如果从另一个目录运行 Express 应用程序，那么对于提供资源的目录使用绝对路径会更安全：

app.use('/static', express.static(_dirname + '/public'))

app.route()

使用 app.route() 为路由路径创建可链接的路由处理程序

app.route('/book')
      .get(function(req, res) {
        res.send('Get a random book');
      })
      .post(function(req, res) {
        res.send('Add a book');
      })
      .put(function(req, res) {
        res.send('Update the book');
      });

在指定目录下创建一个birds.js文件，写入：

express.router()，微型应用程序

var express = require('express');
var router = express.Router();
router.use(function timeLog(req, res, next) {
  console.log('Time: ', Date.now());
  next();
});
router.get('/', function(req, res) {
  res.send('Birds home page');
});
router.get('/about', function(req, res) {
  res.send('About birds');
});

module.exports = router;

接着创建一个路由器模块，写入：

var birds = require('./birds');
app.use('/birds', birds);

这个可以处理 /about 请求。
