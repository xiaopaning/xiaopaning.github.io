line-height

Veröffentlicht am 2017-11-06

之前面试有问过 line-height ，当时直接说不太了解。其实仔细想想，还是因为自己平时工作中 ==不善于观察以及总结== 导致的。

在 css 中输入 line-height 有如下几个值：

  • inherit—->继承父元素。
  • normal —-> 1.2（一般通值）
  • 百分比 —-> 比如：120%
  • 数值（px, em）—->比如： 20px
  • 纯数字（1.5）无单位 —->比如： 1.5

inherit值

如下代码：

//style
#wrap
    font-size: 20px;
    line-height: XXX
}
h1{
    font-size: 18px
}
p{
    font-size: 13px
}
//html
<div id='wrap'>
    <h1>我是标题</h1>
    <p>abcd</p>
</div>

如果 p 标签的父元素 #wrap 的 line-height 值为：

  1. inherit，#wrap—>父元素, p—>#wrap, h1—>#wrap
  2. normal，#wrap—>201.2, p—>131.2, h1—>13*1.2
  3. 百分比(比如120%)，#wrap—>18120%， p—> 18120%， h1—>18*120%, p、h1与父元素的行高相等
  4. 数值(24px)， #wrap—>24px， p—>24px，h1—>24px, p 与父元素的行高相等
  5. 纯数字(1.5)，#wrap—>201.5， p—> 131.5, h1—>18*1.5。

    box分类

  • containing boxs 包含了其他的boxs

  • inline boxs

    The <em>emphasis</em> element is defined as inline


    em元素就是inline boxs，其他的没有特别标签的称为匿名inline boxs

  • line boxs

    inline boxs 在 containing boxs 一个接一个组成line boxs

  • content area

    是围绕着文字看不见的一种box，他的高度为font-size

    inline box 与 line-heigh

  1. line-height和font-size的差别

    {

    font-size: 16px;
line-height: 20px;
// 差别 4px ，即行间距

    }

  2. 拆分一个行间距，来形成一个半行间距。

    4px行间距／2 = 2px半行间距

  3. 半行间距会被应用在 content area 的顶部和底部
    x

  4. content area 包裹着 inline box，而半行间距位于 content area 上部和下部

    5.有时候 inline box 会小于 content area，举个例子，如果line-height 小于font-size，inline box会优先于行高。

    {

    font-size: 16px;
line-height: 12px;
//inline box是12px

    }
    content area会溢出，inline box的顶部和底部会折叠起来，以保证inline box的高度。

    line boxs

    line boxs 的告诉取决于他内部最高的inline box的高度，可以是：

  • 匿名 inline box
  • 增加了line-height 的 inline box
  • 更大的font-size的 inline box
  • 存在上下标的 inline box
  • 存在可替换元素的 inline box

    line box 在containing box宽度内靠彼此的顶部堆积
