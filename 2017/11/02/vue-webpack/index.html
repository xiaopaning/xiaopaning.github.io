vue + vuex + vue-router + webpack

建立根目录

  • 根目录下 npm init 创建package.json配置文件
  • 根目录下 建立dist以及src文件 分别存放 webpack编译后的代码 以及 项目源码

  • 根目录下 建立index.html文件作为 入口文件 ， 代码如下

          <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <meta http-equiv="X-UA-Compatible" content="ie=edge">
      <title>Document</title>
    </head>
    <body>
      <div id='app'>message</div>
      <script src='dist/main.js'></script>
    </body>
</html>

  • 在 src 下建立 main.js, 代码如下：

     // import...from的语法是ES6的，需要用到babel，后面再说
// require的语法是Commonjs的，webpack已经实现了，可以直接使用
const Vue = require('vue')
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue.js!'
    }
 })

安装模块

安装Vue：

npm install vue --save

安装Webpack：

npm install webpack --save-dev

  • 使用webpack编译打包
    在package.json文件中添加一行代码，如下：

    {
  "name": "my-project",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  <!--注意以下代码的修改-->
  "scripts": {
      "test": "echo \"Error: no test specified\" && exit 1"
    },
  "author": "",
  "license": "ISC",
  "dependencies": {
    "vue": "^2.5.2"
  },
  "devDependencies": {
    "webpack": "^3.8.1"
  }
}

    然后运行 npm run server， 浏览器中打开页面。但是会发现启动文件之后，会报错，估计是版本升级的原因。将 main.js文件引入的vue代码替换一下：

    import Vue from 'vue/dist/vue.common.js'

    然后在运行，就可以了。

  • webpack.config.js文件
    根目录建立 webpack.config.js 文件，写入如下代码：

    module.exports = {
  entry:  './src/main.js',
  output: {
    path: __dirname + "/dist",//打包后的文件存放的地方
    filename: "bundle.js"//打包后输出文件的文件名
  }
}

    在 package.json 文件中修改代码，如下

    {
  "name": "my-project",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "webpack",
  },
  "author": "",
  "license": "ISC",
  "dependencies": {
    "vue": "^2.5.2"
  },
  "devDependencies": {
    "webpack": "^3.8.1"
  }
}

    运行 npm start 。达到了同样的效果。

构建本地服务

1.
npm install –save-dev webpack-dev-server
webpack配置文件修改代码

const webpack = require('webpack');
const HtmlWebpackPlugin = require('html-webpack-plugin');
module.exports = {
    ..
  devServer: {
    contentBase: "./dist",//本地服务器所加载的页面所在的目录
    historyApiFallback: true,//不跳转
    inline: true,//实时刷新
    hot: true
  }
}

package.json文件：

{
..
"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "webpack-dev-server --inline --hot",
    "server": "webpack-dev-server --open"
      },
..
}

启动npm start，localhost:8080,就可以看到页面。如果没有看到，不要惊慌，看下原因，如果是浏览器显示 get error字样，哈哈。很简单，如下步骤：

2.

npm install --save-dev html-webpack-plugin

webpack配置文件添加代码： 

   const webpack = require('webpack');
  const HtmlWebpackPlugin = require('html-webpack-plugin');
  module.exports = {
  ..
   plugins: [
       new HtmlWebpackPlugin({
      inject: true,
      filename: 'index.html',
      template:'template/index.html'
      })
]

最后一步，在根目录下建立template文件夹，将之前建立好的index.html移到这个文件内。重新运行，就可以了。

3。引入HotModuleReplacementPlugin热插件

完整的webpack配置文件应为：

const webpack = require('webpack');
const HtmlWebpackPlugin = require('html-webpack-plugin');
module.exports = {

  entry:  './src/main.js',
  output: {
    path: __dirname + "/dist",
    filename: "bundle.js"
  },
  devServer: {
    contentBase: "./dist",//本地服务器所加载的页面所在的目录
    historyApiFallback: true,//不跳转
    inline: true,//实时刷新
    hot: true
  },
  plugins: [
    new webpack.HotModuleReplacementPlugin(),
    new HtmlWebpackPlugin({
        inject: true,
        filename: 'index.html',
        template:'template/index.html'
    })
  ]

}

webpack顺利起个服务就解决了。

配置路由 vue-router

安装： 

npm install --save vue-router

在src下创建 router、 vue 以及 components 文件。
router下创建 index.js以及routes.js

routes.js文件

import Home from '../views/Home.vue'
export default [
  {
    path: '/',
    component: Home
  }
]

index.js文件

import Vue from 'vue/dist/vue.common.js'
import Router from 'vue-router'
import routes from './routes'
import Home from '../views/Home.vue'
Vue.use(Router)
const router = new Router({
  routes
})
export default router

views下创建 Home.vue文件

Home.vue 文件

<template>
  <div><hello></hello></div>
</template>
<script>
  import Hello from '../components/Hello.vue'
  export default {
    components: {Hello}
  }
</script>

components下创建 Hello.vue

//子组建
<template>
  <div>{{message}}</div>
</template>

<script>
  export default {
    data: () => ({message: 'Hello Vue.js!'})
  }
</script>

别忘记修改 main.js 已经 index.html 文件
main.js

import Vue from 'vue/dist/vue.common.js'    import router from './router/index.js'
const app = new Vue({
  router
}).$mount('#app')

index.html

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Demo</title>
</head>
<body>
  <div id="app">
    <router-view></router-view>
  </div>
</body>
</html>

完事了，运行。大功告成。如果这块路由有点乱。仔细研究下这个代码： 

html：
<div id="app">
    <h1>Hello App!</h1>
    <p>
      <router-link to="/user/foo">Go to Foo</router-link>
      <router-link to="/user/bar">Go to Bar</router-link>
    </p>
    <router-view></router-view>
</div>
js:
    //1.定义模版
    const User = {
    template: '<div>User {{ $route.params.id }}</div>'
  }
     //2. 定义路由，并创建 router 实例
  const router = new VueRouter({
    routes: [
      // 动态路径参数 以冒号开头
      { path: '/user/:id', component: User }
    ]
  })
  //2-1.2⃣️步可以将定义路由与创建 router 实例分开.如下
          var routes = { path: '/user/:id', component: User };
          const router = new VueRouter({
            routes
          })
  //3. 创建和挂载根实例。
  const app = new Vue({
    router
  }).$mount('#app')

万变不离其宗，仔细琢磨下每一步的意义，即使路由在绕来绕去，也都能找到根。

⚠️在这里来一个小插曲就是之前。在main.js以及路由js文件中引入vue是通过

import Vue from 'vue/dist/vue.common.js'

这个方法引入的。那是因为版本的问题。想要通过如下方法引入：

import Vue  from 'vue'

我们需要改一下webpack.config.js文件。

const webpack = require('webpack');
const HtmlWebpackPlugin = require('html-webpack-plugin');
module.exports = {
  entry:  './src/main.js',
  output: {
      ..
  },
  devServer: {..},
  plugins: [..],
  module: {..},
  <!--添加的是一下代码-->
 resolve: {
   alias: {
     'vue$': 'vue/dist/vue.esm.js'
   }
 }
}

这样就解决了。下来我们继续来看。

配置 Vuex

跟着我的👣啊，慢呀慢呀走👣👣👣👣👣👣。

npm install vuex --save

哈哈，安装下来了，接下来在建立一大堆的文件，声明： 用vuex做一个加减法。

创建store文件

  1. 建立入口文件 index.js

    //入口问价就是引入需要的文件，把一个个小文件集合起来，暴露给下一个文件，也就是我们这里的 *mian.js*  文件
import Vue from 'vue'
import Vuex from 'vuex'
import * as actions from './actions'
import * as mutations from './mutations'
Vue.use(Vuex)
const store = new Vuex.Store({
  state: {
    count: 0
  },
  actions,
  mutations
})
if (module.hot) {
  module.hot.accept([
    './actions',
    './mutations'
  ], () => {
    store.hotUpdate({
      actions: require('./actions'),
      mutations: require('./mutations')
    })
  })
}
export default store

  2. 建立 action.js

    export const increment = ({ commit }) => commit('increment')
export const decrement = ({ commit }) => commit('decrement')

  3. 建立 mutations.js

    export const increment = state => {
  state.count++
  state.history.push('increment')
}

export const decrement = state => {
  state.count--
  state.history.push('decrement')
}
  4. 一般会建立一个 type.js 文件，用来存放初始值，我们这里先忽略。

哈哈,vuex这样就可以了，千万不要忘记我们的入口文件是 main.js ,所以想应用它，得先把他引入。

main.js

import Vue  from 'vue'
import router from './router'
import store from './store/store.js'
import Hello  from './components/Hello.vue'
const App = Vue.extend({store})
new Vue({
  el: '#app',
  store,
  render: h => h(Hello)
})

components/Hello.vue

 <template>
  <div>
    <p>{{message}}</p>
    <p>click count: {{$store.state.count}}</p>
    <!--可以直接调用引入的action-->
    <button @click="increment">increase</button>
            <button @click="decrement">decrement</button>
  </div>
</template>

<script>
  import { mapGetters, mapActions } from 'vuex'
 // 引入action
  export default {
     data: () => ({
      message: '我咋不会'
    }),
    methods: mapActions([
      'increment',
      'decrement'
    ])
  }
</script>
<style>
  button{
    background: #fff
  }
</style>

可以了，运行一下下。行了吧。开心。。
