JS正则表达式简介与工具使用

https://regexper.com/#b

通配符： *

匹配邮箱： 

^([a-zA-Z0-9_-])+@([a-zA-Z0-9_-])+((\.[a-zA-Z0-9_-]{2,3}){1,2})$

RegExp对象

字面量

var str = 'he is a good boy, she is a good gril'
var reg = /\bis\b/g;
console.log(str.replace(reg, 'IS'))

构造函数

var str = 'he is a good boy, she is a good gril'
var reg = new RegExp('\\bis\\b', 'g')
console.log(str.replace(reg, 'IS'))

修饰符

  • g: gobal

  • i: 大小写

    var str = 'he is a good boy, she Is a good gril'
var reg = /\bis\b/gi;
console.log(str.replace(reg, '0'))
  • m: 多行搜索

字符类

一般情况下正则表达式的一个字符对应字符串中的一个字符

表达式ab\t的含义是 字母：ab + 水平制表符：’tab’

字符类

  1. 我们可以使用一个元字符[]来构建一个简单的类

  2. 所谓的类就是指符合某些特征的对象，一个泛指，而不是特指某个字符

  3. 如表达式[abc]把字符a或者b或者c归为一类，表达式可以匹配这类的字符。

    var str = 'a1b2c3'
var reg = /[abc]/g
console.log(str.replace(reg, '0'))

    字符类取反

  4. 使用元字符 ^ 创建反向类／负向类

  5. 反向类意思是不属于某类的内容

3.[^abc]意思是不是a或b或c的内容

var str = ‘a1b2c3’
var reg = /[^abc]/g
console.log(str.replace(reg, ‘0’))

范围类

[a-z] [a-zA-Z] [0-9]

 var str = '2017-11-15'
var reg = /[0-9-]/g;
var newStr = str.replace(reg, '.')
console.log(newStr)

注意⚠️上述的 - 是在范围内加的 -

JS预定义类及边界

字符 等价 意义
. [^\r\n] 出了回车换行之外的所有字符
\d [0-9] 数字字符
\D [^0-9] 非数字字符
\s [\t\n\xOB\f\r] 空白符
\S [^\t\n\xOB\f\r] 非空白符
\w [a-zA-Z_0-9] 单词字符包括字母数字下滑线
\W [^a-zA-Z_0-9] 非单词字符
预定义类

匹配 ab+数字+.
ab\d.

边界
字符 含义
^ 以xx开始
$ 以xx结束
\b 单词边界
\B 非单词边界

^ 在[]里边是非，在之外是以xx开始

var str = '@12@34'
var reg = /^@/g;
var newStr = str.replace(reg, 'Q')
