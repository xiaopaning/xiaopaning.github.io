###CSS变量（自定义属性）
平时开发当中使用sass比较多，开发方便，易于维护。最近css开始添加一些动态特性，并且已有大量的浏览器支持，下来我就试着玩一下动态css

1.自定义css动态属性：–

:root {
  --main-color: #06c;
  --accent-color: #0ff;
  --MAIN-COLOR:#f0f0f0;
}
 h1 {
    color: var(--main-color);
}
h2{
    color: var(--MAIN-COLOR);
}

值得注意⚠️： 动态属性名是有大小写之分的。

--foo: if(x > 5) this.width = 10%;

  1. CSS变量是级联的

    :root { --color: blue; }
div { --color: green; }

.alert { --color: red; }

*{ color: var(--color); }
<p>I inherited blue from the root element!</p>
<div>I got green set directly on me!</div>
<div class='alert'>
  While I got red set directly on me!
  <p>I’m red too, because of inheritance!</p>
</div>
